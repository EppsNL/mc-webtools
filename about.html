What is MC Webtools and who is it for?

MC Webtools is a personal tool I made to simplify my Minecraft playing workflow. Instead of having multiple windows open for my notes, locations, map and portal calculations, everything I need is now accessible from a single browser window. New features may be added once I run into additional use-cases for this tool that would benefit my personal play style.

Availability and Data Policy

Availablity of the online version is not guaranteed and I offer no support for when (not if) things break. All data is stored client-side in the browser's local storage. No data you enter gets send or stored to the webserver. To prevent accidents while cleaning your browser storage, I strongly suggest making frequent backups using the export data feature in the footer. Exported data is presented as json, which can be read through Notepad.
